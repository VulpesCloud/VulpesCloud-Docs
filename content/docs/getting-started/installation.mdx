## Requirements
To run VulpesCloud, you need to have the following software installed on your system:
- **[Java 21](https://adoptium.net/en-GB/temurin/releases/?version=21)** or newer
- **screen**

## Installation
1. **Download the latest release** from the [VulpesCloud GitHub repository](https://github.com/VulpesCloud/VulpesCloud/releases).
2. **Extract the downloaded archive** to a directory of your choice.
3. **Run the start.sh or start.bat** depdending on your OS.
4. **Follow the setup instructions** to complete the setup process.
